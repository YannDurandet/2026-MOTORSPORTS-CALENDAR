## Naming Convention
- `{venue}.svg` — generic / single-series
- `{venue}-motogp.svg` — MotoGP uses different circuit config
- `{venue}-fe.svg` — Formula E street overlay
- `{venue}-wsbk.svg` — WSBK uses different circuit config
- `{venue}-oval.svg` — superspeedway oval layout
- `{venue}-road.svg` — road course at an oval venue
- `{venue}-f1.svg` — F1-specific only when MotoGP also uses same venue
- `{location}-wrc.svg` — all WRC / ERC national flags
